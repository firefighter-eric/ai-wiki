# Veo 3.1

## 简介

Veo 3.1 是 Google DeepMind 当前视频生成主线的最新官方节点。在当前知识库里，它代表 Google 把 cinematic video generation、native audio 与参考控制接入 `Gemini / Flow` 产品生态的路线。

## 关键属性

- 类型：视频生成模型
- 代表来源：
  - [Google DeepMind - 2026 - Veo](../../wiki/summaries/Google%20DeepMind%20-%202026%20-%20Veo.md)
- 当前角色：Google 视频生成主线代表节点

## 相关主张

- Veo 3.1 的竞争重点是 realism、prompt adherence、physics 和 audio-video alignment。
- 它不是孤立实验模型，而是被明确放进 `Gemini`、`Flow` 与 API 构建场景中。
- 从当前来源能确认 Veo 强调参考图像控制与原生音频，但方法细节仍未公开充分披露。
