# DeepSeek

## 简介

DeepSeek 是当前知识库中连接 `DeepSeek-V3`、`DeepSeek-R1` 与 `DeepSeek-V4` 的模型家族概念页。它同时覆盖高效预训练骨干、面向推理能力的强化学习路线，以及百万上下文与 agent-oriented 的效率架构路线。

- 当前角色：连接 `LLM预训练`、`LLM RL`、长上下文效率与 agent 工作流的代表性系列

## 相关主张

- `DeepSeek-V3` 体现的是 MoE、训练效率和强开源模型能力。
- `DeepSeek-R1` 体现的是“用 RL 直接激励推理能力”，并进一步延伸到蒸馏模型。
- `DeepSeek-V4` 则把 DeepSeek 的效率叙事推进到 `1M` 上下文、混合 attention、KV cache 压缩与长程 agent 任务。
- 因此在当前知识库里，DeepSeek 不是单一模型名，而是“预训练骨干 + 推理强化学习 + 长上下文 agent 效率架构”的系列入口。
